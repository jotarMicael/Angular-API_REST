# Front(Angular)-Back(Java)

### Pre-requisitos 📋

```
Instalar las dependencias tanto en Angular como en el proyecto de Java

```

### Instalación 🔧

```
Angular: npm install

Java: Update Maven Project

```

## Despliegue 📦

```
Angular: ng serve

Java: levantar el proyecto con tomcat v9 o inferiores

```
